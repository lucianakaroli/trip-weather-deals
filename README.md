# trip-weather-deals

API em Go para recomendação de destinos com clima favorável e passagens baratas.

## Funcionalidades
- Consulta mockada de clima
- Consulta mockada de passagens
- Recomendação de destinos sem chuva e dentro de um preço máximo
- Testes automatizados
- Pipeline CI com GitHub Actions
- Infraestrutura como código com Terraform

## Executando localmente

## Comandos com Makefile

```bash
make help
make test
make fmt
make vet
make build
make run
```

```bash
go run ./cmd/api
```
